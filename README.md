# Insect Museum Landing Page

This project is a responsive museum landing page built using React and Tailwind CSS.

## Features
- Responsive Navbar
- Hero Section
- About Section
- Experience Cards
- Footer
- Scrolling Announcement Bar
- Automatic Day/Night Mode

## Day/Night Mode

The theme automatically switches between day and night using React state and setInterval().
The mode changes every 5 seconds.
